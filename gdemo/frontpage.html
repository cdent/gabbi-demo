Gabbi Demo Application

This is the front page of the Gabbi Demo Application!

/
  • GET the front page
  • POST to create a container
/id
  • GET a container
  • PUT to update a container
  • DELETE a container and its content
/id/objects
  • GET a list of objects in a container
  • POST to create an object in a container
/id/objects/id
  • GET an object
  • PUT to update an object
  • DELETE an object
